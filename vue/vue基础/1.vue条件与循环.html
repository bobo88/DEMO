我们用 v-for 指令根据一组数组的选项列表进行渲染。 v-for 指令需要以 item in items 形式的特殊语法， items 是源数据数组并且 item 是数组元素迭代的别名。

循环数组：

  • {{ item }}
  • {{ index }} ---> {{ item }}

在 v-for 块中，我们拥有对父作用域属性的完全访问权限。 v-for 还支持一个可选的第二个参数为当前项的索引（在源数据为数组或者对象时）。

循环对象：

  • {{ item }}
  • {{ index }} ---> {{ key }} ---> {{ item }}

在 v-for 块中，如果源数据是对象的话，可以传三个参数，第一个依然是对象属性(或者方法)迭代的别名，提供第二个的参数为键名，第三个参数为当前属性(或者方法)的索引。
